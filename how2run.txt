Assuming java is java17+ jdk with javaFX modules (provided by Azul for example):
java -cp "./modules/lib/*" -p ./modules -m xsdexplorer/com.xsdexplorer.XsdExplorer

On Debian based systems (such as Ubuntu), you will have to install the openjfx package.
And run using: java --module-path "/usr/share/openjfx/lib:./modules" -m xsdexplorer/com.xsdexplorer.XsdExplorer
